# my_app

## Project setup
```
vue create my_app
-> Manually select features
-> 空格键选择 需要安装的Router、Vuex、CSS Pre-processors、linter / Formatter（选择完按enter）
-> 3.x (Preview)
-> history mode yes
-> Sass/SCSS ( with dart-sass )
-> ESLint + prettier
-> lint on save && lint and fix on commit
->...
```
